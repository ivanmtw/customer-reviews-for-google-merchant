Embeds Google Customer Reviews code in your website to collect customer feedback

== Description ==

Please note that Google Customer Reviews are not the same as Google My Business Reviews.

Google Customer Reviews is a product that enables sellers to collect valuable feedback from customers whoâ€™ve made a purchase on their site. Customers can choose to receive a survey after order delivery where they rate their experience with just one click.

'Customer Reviews for Google Merchant' Plugin add tracking code from Google Merchant Center to add customer
feedback functionality on your website

You can see all offered surveys from your customers in your Merchant account (https://merchants.google.com/mc/customerreviews/overview).

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `customer-reviews-for-google-merchant` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set reqired parameters in WooCommerce Integration tab `/wp-admin/admin.php?page=wc-settings&tab=integration&section=customer-reviews-for-google-merchant`
